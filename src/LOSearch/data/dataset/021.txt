Java Applets for a range of algorithms taken from Papatheodorou's book "Introduction to Algorithms designing and analysis". At the time of cataloging the following algorithms were covered: Project Order Solution Using Greeding, Minimum Genetic Tree Finder Using Prim and Kruskal Algorithms, Minimum Genetic Tree Using Prim Algorithm and adjoining array, Minimum Genetic Tree Finder Using Kruskal Algorithm, Minimum Route Finder Using Dijkstra Algorithm, All Minimum Routes Finder (Floyd Algorithm), and Queens Problem Solver Using Backtracking.
